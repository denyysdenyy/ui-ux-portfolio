Работаем поэтапно, сверху вниз, по одному компоненту за раз. Не делать всё сразу.
В процессе задавать уточняющие вопросы.

**Why:** Пользователь хочет внимательный, контролируемый процесс с возможностью давать фидбек на каждом этапе.

**How to apply:**
- Брейкпоинты: стандартные Tailwind (sm:640, md:768, lg:1024, xl:1280, 2xl:1536)
- Container: px-4 на мобильных
- Font sizes: через clamp()
- Мобильное меню: пока не нужно, просто убираем навигацию
- Benefits: адаптируем анимацию (уменьшаем карточки)
- Memes на мобильных: Заголовок → Слайдер → Видео → Кнопка
- Works: постепенно уменьшать карточки, на md или меньше — 1 колонка
- Section margins: 300px → 130px на мобильных (через clamp)
- Есть Figma-макет, будет предоставлен
